1. Introdução

Bem-vindo ao SoundRTS!

SoundRTS é um jogo de estratégia em tempo real inspirado nas séries Warcraft e Starcraft, mas trata-se de um audiogame, Por tanto adaptado para cegos. Nesse jogo, você vai explorar as imediações, explorar Minas de ouro e bosques, construir edifícios, recrutar operários e soldados, e lutar contra o inimigo!

O jogo está pré-configurado no modo de mapa: você está voltado para o norte, e pode selecionar todos os objetos sem necessidade de se virar. Mas você pode também jogar no modo de primeira pessoa, útil na exploração e no ataque.

2. Tutorial

Este tutorial vai ajudá-lo a aprender os principais controles do jogo.

2.1 Capítulo 1 tutorial

Se você usa Jaws ou outro leitor de tela, será necessário desativá-lo para poder acessar o teclado.

Quando o jogo começar, você vai ouvir o nome do quadrado onde está: "A1". A presença de uma mina de ouro em A1 também será mencionada. Para examinar este
quadrado, pressione a tecla tab algumas vezes. Você vai notar que este quadrado tem uma única saída (um caminho), uma mina de ouro, um bosque, o Edifício
Principal 1 (é a sua base), e o operário 1.
Então pressione"Page Down" para saber se outro quadrado pode ser examinado. Vai ouvir então o que existe em "C1", (pressione"Tab" para saber isto) a
fazenda, o caminho, o bosque e o soldado 1.
Se pressionar "Page down" novamente, estará de novo em "A1". Isso significa que apenas 2 quadrados Podem ser examinados por agora.
Outro método para se mover no mapa é utilizar as setas direcionais. Nesse modo, pode saltar por cima de quadrados desconhecidos para dar uma ordem de movimentação
para um quadrado desconhecido.
O objetivo dessa missão é construir uma fazenda e um quartel. Somente os operários podem construir. Mas para isso precisa de ouro e madeira e também de
um espaço livre (um terreno). Para saber quanto possui de ouro, madeira e comida, pressione Z, X e C respectivamente.
Vamos então pedir ao operário para ir recolher ouro. Primeiro, para o controlar, pressione Q até ouvir "Operário 1, esperando ordens!". Em seguida, pressione
A até ouvir "explorar"; então pressione Tab até selecionar a mina de ouro e por fim pressione Enter para confirmar. O operário irá iniciar o recolhimento
de ouro.
Para ir mais rápido, vai precisar de mais operários. Pressione Q até controlar o Edifício Principal. Então pressione A até selecionar "recrutar operário"
e pressione Enter. Após alguns segundos, Um novo operário (Operário 2), irá aparecer.
Para pedir ao Operário 2 que recolha madeira, faça como o primeiro, ou para ir mais rápido pressione Q até controlar operário 2, em seguida Tab para selecionar
a mina de ouro, então pressione "Backspace". Essa tecla vai ativar a ação padrão no alvo selecionado: Neste caso, a ação padrão é "explorar a mina de ouro".
Antes de recrutar o terceiro operário convém informar o Edifício Principal 1 que os novos operários que serão recrutados, terão a missão de explorar essa
mina de ouro. Para isso, pressione Q até controlar o Edifício Principal 1. Em seguida, pressione Tab até selecionar a mina de ouro. Finaumente pressione
"Backspace" Para tornar a ordem de explorar a mina de ouro uma configuração. Feito isso, só precisa pressionar A Para escolher "recrutar operário", e Enter
para confirmar.
Quando tiver ouro suficiente ou a mina de ouro ficar esgotada, pressione D para controlar Todos os operários em seguida Tab até selecionar o bosque. Por
fim pressione "Backspace" para ativar a exploração do bosque.
O resto não é difícil.

2.2 Capítulo 2 tutorial
Dica: Agrupe as suas unidades de combate. Para fazer isso, controle-as(com a tecla S) e mande-as Patrulhar entre os quadrados que quer proteger.

2.3 Dicas e Sugestões

  • Mantenha suas tropas ativas.
  • Mande seus soldados patrulhar. Uma patrulha militar pode proteger uma zona ampla mantendo as suas tropas em atividade.
  • Use pontos de convergência para seus edifícios. Os edifícios que recrutam unidades podem definir pontos de convergência. As novas unidades irão visar este novo ponto. Por exemplo, um novo operário irá explorar uma mina de ouro definida como ponto de convergência.
  • Use o modo defensivo para exploração. Uma unidade em modo defensivo irá fugir se encontrar unidades inimigas mais fortes. Dessa forma você poderá saber quantos inimigos se encontram num determinado quadrado sem sacrificar as suas unidades que estão explorando. Os operários estão pré-configurados em modo defensivo, mas os guerreiros também podem ser alterados para esse modo.
  • Uma unidade que fugiu, esquece as suas ordens. Por exemplo, um operário que esteja recolhendo ouro, não irá trabalhar depois de fugir, uma vez que ele fugiu e sobreviveu! Terá que lhe dar as ordens novamente para ele voltar a recolher ouro.

3. Guia de Referência

3.1 Lista de comandos

O jogo consiste em dar ordens às suas unidades e edifícios. Para dar ordens A uma unidade, em primeiro lugar você deve controlá-la. Se você pressionar F10 durante o jogo, irá para o menu. Se utiliza Jaws ou outro leitor de tela, será necessário desativá-lo para poder acessar o teclado.

Movendo-se pelo Mapa

As setas direcionais fazem com que se mova pelo mapa de um quadrado para outro. Se existir um caminho direto entre o quadrado atual e o novo quadrado, irá ouvir o som correspondente Dependendo do tipo de passagem (caminho ou ponte). Se não existir nenhuma passagem, irá ouvir o som de uma colisão e ficará no mesmo quadrado (pressione control + setas direcionais para saltar por cima de um obstáculo. Se ainda não souber se a passagem existe (quadrado desconhecido) nesse caso nenhum som será ouvido.

Outra forma de se mover pelo mapa é através da tecla Page Up, que vai levá-lo diretamente para o próximo quadrado de interesse Sem passar pelos quadrados vazios.

Desde o SoundRTS 1.1 alpha 3, algumas variações estão disponíveis:
  • pressione Alt + PageUp/PageDown para selecionar o quadrado desconhecido anterior ou posterior respectivamente
  • pressione Shift + PageUp/PageDown para selecionar respectivamente o quadrado anterior ou posterior que contenha recursos

Quando você estiver controlando uma unidade e pressionar a barra de espaços, irá automaticamente segui-la quando ela se mover de um quadrado para outro.

Escolhendo uma Unidade a Controlar

Para controlar a próxima unidade local, pressione Q.

Para controlar o próximo edifício, pressione W. Para controlar o próximo operário, pressione E. Para controlar todos os operários no quadrado atual, pressione D. Para controlar o próximo operário inativo pressione Alt E. Para controlar todos os operários inativos presentes no quadrado atual, pressione alt D. Para controlar o próximo soldado pressione R. Para controlar todos os soldados presentes no quadrado atual, pressione F. Para controlar o próximo arqueiro, pressione T. Para controlar todos os arqueiros presentes no quadrado atual, pressione G. Para controlar o próximo cavaleiro, pressione Y. Para controlar todos os cavaleiros presentes no quadrado atual, pressione H. Para controlar a próxima catapulta, pressione U. Para controlar todas as catapultas presentes no quadrado atual pressione J. Para controlar o próximo dragão, pressione I. Para controlar todos os dragões presentes no quadrado atual, pressione K. Para controlar o próximo mago, pressione O. Para controlar todos os magos presentes no quadrado atual, pressione L.

As teclas que servem para controlar a próxima unidade, se forem combinadas com Shift fazem com que controle a unidade anterior! Por exemplo, para controlar o operário anterior pressione Shift + E.

Para controlar todas as unidades presentes no quadrado atual do mesmo tipo da atuaumente controlada, pressione1. Para controlar a metade ou um terço pressione 2 ou 3. Para deixar de controlar um grupo, pressione0.

Para controlar todas as unidades de combate presentes no quadrado atual pressione S. Para controlar todos os operários e unidades de combate presentes no quadrado atual pressione Alt + S. As teclas que representam o controle dum grupo de unidades presentes no quadrado atual se forem combinadas com a tecla Control, vão controlar todas as unidades desse grupo presentes em todo o mapa. Por exemplo, para controlar todos os Soldados, pressione Control + R ou Control + F.

Dando Uma Ordem: Método Principal

Para dar uma ordem a uma unidade controlada, o principal método consiste em escolher a ordem na lista e escolher o alvo se a ação assim o requerer.

Para escolher a ordem dentro da lista, pressione A (e Shift A) para selecionar a ordem.

Se tiver necessidade de selecionar um alvo, pressione Tab (e Shift Tab) para selecionar o alvo. Para selecionar um quadrado remoto como alvo, use as setas direcionais.

Pressione Enter para confirmar sua seleção.

Dando Uma Ordem: Método alternativo, Mais Rápido

O Segundo método para dar uma ordem consiste em selecionar o alvo com Tab (ou as setas direcionais) e então deverá pressionar Backspace. A ordem padrão será dada. Por exemplo, um operário que esteja numa mina de ouro, irá explorá-la se você pressionar Backspace.

Examinando a Situação

Para obter informação sobre a unidade controlada (ou o grupo controlado), pressione a Barra de espaços. Nesse caso você irá para o quadrado ocupado pela unidade (ou pelo líder do grupo).

Para saber quanto ouro você tem, pressione Z. Pressione X para a madeira, pressione C para a comida.

Para saber o estado de saúde da unidade atuaumente controlada pressione V.

Para examinar novamente um objeto selecionado com Tab, pressione Control.

Dando Uma Ordem Sem Cancelar as Anteriores (enfileiramento de comandos)

Segure a tecla Shift antes de pressionar Enter para confirmar a ordem.

Isso também funciona para ordens padrão: Segure a tecla Shift antes de pressionar Backspace.

Pressione espaço para verificar se a unidade tem outras ordens para executar.

Aqui estão algumas situações em que o enfileiramento de comandos pode ser útil:

  • Exemplo 1: você quer que um operário explore cada quadrado inicial do mapa (para saber onde está o inimigo). Controle o operário, mova o cursor para um quadrado inicial e, em seguida, pressione Shift + Backspace. Repita o procedimento para cada quadrado inicial.
  • Exemplo 2: você quer que um operário explore diversos recursos. Selecione um recurso, pressione Shift + Backspace. Repita para o próximo recurso. Quando um recurso é esgotado, o operário vai explorar o próximo.

Dando Uma Ordem Imperativa

Se você segurar a tecla Control ao pressionar Enter ou backspace, a ordem será imperativa.

Unidades com uma ordem imperativa tendem a ignorar tudo o que não seja diretamente relacionado com a ordem. Se eles tiverem que ir a algum lugar e passarem pelas unidades inimigas, simplesmente vão ignorá-las. Isto é perigoso na maioria das vezes, mas pode ser útil em alguns casos, por exemplo, para atingir um objetivo muito importante.

Aqui estão alguns exemplos em que um comando imperativo pode ser útil:

  • Exemplo 1: você deseja que suas unidades ataquem um inimigo específico, edifício ou unidade, e ignore o resto. Selecione o destino e pressione Control + Backspace.
  • Exemplo 2: você deseja retirar suas unidades de um lugar e ignorar as unidades inimigas. Selecione outro quadrado e pressione Control + Backspace.

Você pode enfileirar vários comandos imperativos. Segure as teclas Control + Shift em vez de Shift.

Atacando Uma Unidade Amiga

Escolha a unidade de destino e pressione Shift Backspace: as unidades vão atacar o alvo.

Exceção: se o alvo for um edifício danificado (ou uma unidade reparável como uma catapulta) e você estiver controlando os operários, eles irão consertá-la.

Outros comandos

Para ouvir novamente toda a lista de mensagens, use F5 e F6. Para interromper a frase atual, use a tecla Alt.

Para sair do jogo ou acessar o menu do jogo, utilize F10. Alt F4 ou Control C fazem exatamente o mesmo.

Control + espaço, farão com que o jogo passe para o modo de primeira pessoa. Pressione Escapepara voltar ao modo de mapa.

Home e End, ou mais e menos do teclado numérico, para alterar o volume do som. Control Home/End, ou control +/- do teclado numérico: para aumentar/diminuir o volume da voz.

Nos menus as setas direcionais, são usadas para cima e para baixo para selecionar, Direita para confirmar, Esquerda para cancelar.

F3: Informa o tempo de jogo Control F3: Liga/desliga o aviso a cada minuto (O padrão é desligado)

Control Tab seleciona somente bosques, minas de ouro, terrenos e outros objetivos a reparar ou construir (edifícios danificados, catapultas danificadas...).

Alt + R Repete ordens dadas anteriormente. Útil para recrutar novamente unidades do mesmo tipo, ou construir edifícios do mesmo tipo várias vezes. Alt + G Repete ordem anterior, validando-a caso não necessite de parâmetros adicionais. Por exemplo, para recrutar 5 arqueiros adicionais, pressione alt+g 5 vezes.

Desde a versão o 1.1 alpha 3 do SoundRTS, Alt + A é equivalente a Alt + G.

As configurações e ativação de SAPI 5 só podem ser acessadas durante um jogo, mas a configuração pode ser salva.

  • Para Ligar/desligar o SAPI: durante um jogo, pressione Control Alt F12 (o padrão é desligado)
  • pressione Control F12 para selecionar a próxima voz, e Control Shift F12 para selecionar a voz anterior
  • Alterar velocidade SAPI: Alt (Shift) F12
  • pressione Control Alt Shift F12 para salvar a configuração de SAPI
  • volume SAPI (as mesmas teclas de volume para vozes)

3.2 Unidades

Operário

O operário é importante, mas frágil. Os operários são utilizados para recolher ouro e madeira, construir e reparar edifícios. Muitos operários juntos podem ajudar na construção e reparo de edifícios e com isso aumentar a velocidade e tempo de execução. Quando um operário pode construir novos tipos de edifícios, esta nova opção irá aparecer no fim da lista de ordens (selecionar com a Tecla A).

O Operário pode lutar, e pode juntar-se a ataques maciços com o comando Control + Alt + S.

Ele tem4 pontos de vida. Sua arma retira 1,5 pontos de vida a cada um segundo. O alvo deve estar a menos de 0,8 metros de distância. Sua velocidade é de1.5 metros por segundo. Ele custa4 de ouro. O edifício principal é seu local de recrutamento. ele necessita 1 de comida para sobreviver.

  • custo em alimentação: 1 de comida

  • saúde: 4 pontos de vida

Máquina Voadora

Esta nova versão da máquina voadora têm a capacidade de transportar unidades e detectar unidades invisíveis, mas não pode se defender. Unidades transportadas não podem atacar a partir de uma máquina voadora. Se o transporte for destruído, as unidades que estiverem dentro serão destruídas também.

Para carregar unidades:

  • Método 1: controle as unidades a serem transportadas, escolha o destino para a máquina voadora e pressione backspace
  • Método 2: controle a(s) máquina(s) voadora(s), escolha o local para onde devem ser transportadas as unidades e pressione backspace.
  • Método 3: controle a(s) máquina(s) voadora(s), selecione o quadrado onde estão as unidades a serem transportadas, selecione com a letra a a opção "carregar todos de..." e pressione Enter.

Para descarregar as unidades: controle a(s) máquina(s) voadora(s), selecione no menu de ações a opção "descarregar todas em", escolha o destino e pressione Enter.

  • Recrutada na oficina

  • Custo total de alimentação: 1 de comida
  • tempo de recrutamento: 10 segundos
  • saúde: 30 pontos de vida
  • velocidade: 1.5 metros por segundo
  • pode ver os quadrados ao redor
  • detecta as unidades invisíveis

  • tempo de recrutamento: 20 segundos

Por outro lado, a catapulta é muito frágil e não pode sequer defender-se contra uma unidade no mesmo quadrado. Ela se move muito lentamente.

3.4 Habilidades

Conversão

A conversão é uma maneira de tomar o controle de uma unidade inimiga. A unidade passará a ser uma das suas unidades. Edifícios também podem ser convertidos.

Para converter uma unidade inimiga:

  1. escolha quem irá cumprir a ordem
  2. no menu de ações, selecione conversão
  3. Selecione uma unidade ou edifício inimigo como alvo (nota: se você estiver selecionando uma unidade móvel em um quadrado que não esteja sendo visto no momento, a unidade pode estar em outro lugar e a ordem será cancelada quando o encarregado chegar no quadrado)
  4. confirme a ordem com enter
  • requer: pesquisa para conversão
  • custo em mana: 150 pontos de mana

A névoa mortal danifica qualquer unidade reparável no quadrado de destino.

4. Jogos Multiplayer

4.1 Iniciando um Jogo Multiplayer

Selecione "multijogador", e em seguida "selecionar na lista de servidores".

Em um servidor público você tem a possibilidade de organizar um jogo.

Dica: enquanto estiver esperando por jogadores on-line, você pode iniciar outra sessão do SoundRTS e jogar contra o computador. Quando um jogador aparecer, pressione a tecla F10 para pausar seu jogo local e começar a jogar on-line.

4.2 Chat

Você pode conversar apenas na entrada do servidor e na sala de pré-jogo. Porém o organizador do jogo estará temporariamente indisponível durante a seleção do mapa e da velocidade do jogo.

Para dizer algo a todos na mesma sala, tecle S, digite sua mensagem e em seguida pressione enter. Se a função de texto para fala não estiver habilitada, a mensagem será soletrada.

Outra opção seria usar o skype ( ou outro similar).

4.3 Jogar em Equipe

As equipes devem ser criadas antes do início do jogo, pois não podem ser alteradas posteriormente.

Jogadores aliados compartilham o conhecimento do mapa, a vitória, e suas unidades podem ajudar-se mutuamente. O operário pode usar um edifício aliado para armazenar um recurso (o recurso será ainda adicionado ao exército a que pertence esse operário).

Por padrão, em um servidor os computadores não são aliados.

Nota: quando estiver jogando contra o computador pela opção "Jogar sozinho", os computadores são aliados.

Dica: para jogar sozinho com um computador como aliado, execute um servidor privado.

4.4 Como Tornar Seu Servidor Acessível a Outros Jogadores

Aqui vão algumas dicas caso ninguém consiga conectar-se ao seu servidor.

Se você estiver protegido por um firewall, será necessário liberara porta 2500 (em TCP).

No entanto, se você usa um roteador, será preciso adicionar um servidor virtual à porta 2500 para o IP do seu servidor. Sem isto, o roteador não vai saber a qual computador deve direcionar as informações. No caso de um Servidor virtual, você terá necessidade de fornecer um Endereço IP fixo ao seu servidor com DHCP.

4.5 Como Fazer download de um Mapa em um Servidor

Jogue um jogo com o mapa que você gostaria de baixar. após o jogo, o mapa poderá ser copiado da sua pasta temporária with the following name: SoundRTS_recent_map.txt

5. Elementos do Jogo

Comida

O recrutamento de uma nova unidade só pode ocorrer se o jogador tiver comida suficiente. Se algumas fazendas forem destruídas ou novas unidades forem obtidas sem recrutamento, e a comida se tornarem insuficientes, as unidades atuais serão mantidas sem qualquer problema. Porém nenhum novo recrutamento será permitido.

Melhoramentos Técnicos

As melhorias serão aplicadas a todas as unidades atuais e futuras.

Unidades Voadoras

Desde a versão 1.1 do SoundRTS, as unidades voadoras voam diretamente para o objetivo, ignorando os caminhos terrestre.

Unidades Invisíveis e Detectores

Existem ao menos 3 maneiras para se defender de unidades invisíveis:

  • Levar para o local uma unidade detectora ou usar alguma magia para detecção, e finaumente atacar a unidade
  • usar alguma magia destrutiva sobre o quadrado em que estiver a unidade
  • Ignorar as unidades invisíveis e destruir rapidamente todos os edifícios inimigos

6. Mais Detalhes sobre Personalização

SoundRTS.ini

Este arquivo está localizado na Pasta de Usuário. Ele contém diversos parâmetros para o funcionamento do jogo. Quando o SoundRTS for iniciado, se o arquivo SoundRTS.ini contiver algum erro, um novo arquivo será gerado com todos os parâmetros que puderem ser recuperados do arquivo corrompido.

Suporte Experimental a Leitores de Tela

Esta característica requer uma versão do Windows installer (ou a versão multiplatavorma contendo a dll da ScreenReaderAPI).

Para ativar o suporte a leitores de tela, edite o arquivo SoundRTS.ini e substitua "srapi = 0" por "srapi = 1". Não será preciso nenhum procedimento adicional. O SoundRTS deverá usar o leitor de tela assim que iniciado. Para utilizar novamente outro método de saída, restaure a linha para "srapi = 0".

Um parâmetro adicional chamado "srapi_wait" determina quanto tempo o SoundRTS irá esperar antes de enviar a próxima mensagem. Com "srapi_wait =.1" e uma mensagem de 10 caracteres, SoundRTS irá aguardar .1 * 10 = 1 segundo.

Nenhum outro parâmetro de leitores de tela pode ser configurado no jogo.

Suporte Experimental ao Speech Dispatcher no Linux

Esta característica requer a instalação dos pacotes "speech dispatcher" e "speech dispatcher for Python".

Para ativar, execute o SoundRTS utilizando as opções --speechd ou -s na linha de comando. Por exemplo: python soundrts.pyc -s

Pasta Temporária

A pasta temporária contém os logs e os mapas anteriormente utilizados.

Se a pasta principal contiver uma pasta chamada "tmp", essa será a pasta temporária. Se não existir a pasta "tmp", então a localização dependerá do seu sistema operacional:

  • Windows XP: "C:\Documents and settings\seu_nome_de_usuário\Local Settings\Temp"
  • Linux: "home/seu_nome_de_usuário/.SoundRTS unstable/tmp"

Pasta do Usuário

A pasta do usuário contém as configurações do jogo e os mapas personalizados.

Se a pasta principal contiver uma pasta chamada "user", essa será a pasta do usuário. Se a pasta "user" não existir, então a localização dependerá do seu sistema operacional::

  • Windows XP: "C:\Documents and settings\seu_nome_de_usuário\Application data\SoundRTS unstable"
  • Linux: "home/seu_nome_de_usuário/.SoundRTS unstable"

Como Criar uma Versão Portable do SoundRTS

Desde a versão 1.0 beta 10 p do SoundRTS é possível ter todos os arquivos (log, configuração, mapas personalizados...) armazenados na pasta do jogo. Faça o seguinte:

  • instale o SoundRTS
  • No Windows: copie a pasta do jogo de "Arquivos de Programas" para outro local (sua área de trabalho, um drive USB, etc)
  • em sua nova pasta do SoundRTS, crie duas pastas: "tmp" e "user". "user" pode ser uma cópia da sua pasta do usuário (desde que a versão seja a mesma).
  • Windows: para abrir o programa, execute "soundrts.exe".

Como Alterar a Velocidade padrão de um Jogo local

Procure pelo arquivo SoundRTS.ini na pasta do usuário e modifique a linha "speed = ...". Use um número inteiro.

alterando algumas Teclas

Desde a versão 1.1 do SoundRTS, o layout de teclado é definido em um arquivo chamado bindings.txt localizado na pasta do jogo.

Gravando Novas vozes

Se uma voz SAPI 5 estiver disponível, build_tts.exe grava o texto definido no arquivo tts.txt e converte o resultado para o formato ogg requerido.

Criando um app Para Mac

Se você conseguiu instalar a versão multiplataforma no Mac, talvez possa criar um app para facilitar a instalação de outros jogadores. Aqui estão algumas dicas. Use py2app. Você não precisa da fonte. Por exemplo, crie um arquivo chamado soundrts_launcher.py que contenha esta única linha: "import soundrts" Você também pode criar um arquivo chamado server_launcher.py com a linha: "import soundrts". Certifique-se de executar "python soundrts_launcher.py" que esteja funcionando.

Usando o windows Installer em Uma Conta Limitada (Windows XP)

Você pode usar uma conta limitada do Windows XP para instalar o SoundRTS. Basta escolher outra pasta diferente de "arquivos de programas".

Se você instalar o SoundRTS usando uma conta de administrador, é possível jogar através de uma conta limitada..
